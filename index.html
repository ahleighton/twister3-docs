Twister3¶

This is the documentation site for the Twister 3, a fast, open-source microwire twister.

Features¶

  • Fast: Make 70 tetrodes per hour

  • Constant wire tension during twist

  • Automation increases repeatability

  • No need to fold wire before twisting

  • Control box sets twist parameters

  • Easy to assemble
